## Run this project
  git clone https://github.com/andersonassisb/ZenChallenge.git <br />
  cd ZenChallenge <br />
  yarn install <br />

  ### Android
    yarn android

  ### iOS 
    yarn install-pods (for Intel Processors)
    
    sudo arch -x86_64 gem install ffi
    yarn install-pods-m1 (for Apple Silicon)
    
    yarn ios

  ### Run Unit Tests 
    yarn test

## Clearing Android
  yarn clean-android

## Clearing iOS
  yarn clean-ios
